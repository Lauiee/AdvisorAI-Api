# 프론트엔드 디자인 참고 사이트 가이드

## 🎨 일반 디자인 갤러리 사이트

### 1. **Dribbble** (https://dribbble.com)

- **특징**: 전 세계 디자이너들의 작품 공유 플랫폼
- **검색 키워드 추천**:
  - `education platform`
  - `academic dashboard`
  - `counseling app`
  - `graduate school`
  - `AI advisor`
  - `matching system`
- **장점**: 최신 트렌드, 다양한 스타일, 상세한 디자인 설명

### 2. **Behance** (https://www.behance.net)

- **특징**: Adobe 운영, 포트폴리오 중심
- **검색 키워드 추천**:
  - `Educational Technology`
  - `University Platform`
  - `Student Portal`
  - `AI Assistant Interface`
- **장점**: 완성도 높은 프로젝트, 전체적인 플로우 확인 가능

### 3. **Awwwards** (https://www.awwwards.com)

- **특징**: 우수한 웹사이트 디자인 선정 및 소개
- **검색 키워드 추천**:
  - `Education`
  - `University`
  - `Platform`
- **장점**: 실제 구현된 사이트, 인터랙션 확인 가능

### 4. **SiteInspire** (https://www.siteinspire.com)

- **특징**: 카테고리별 웹사이트 갤러리
- **검색 카테고리**:
  - Education
  - Business
  - Portfolio
- **장점**: 카테고리별 분류, 다양한 스타일

### 5. **CSS Design Awards** (https://www.cssdesignawards.com)

- **특징**: 창의적이고 혁신적인 웹 디자인
- **장점**: 최신 기술 활용 사례

## 🎨 UI 컴포넌트 라이브러리 (실제 구현 참고)

### 1. **Material Design** (https://material.io/design)

- Google의 디자인 시스템
- **컴포넌트**: 카드, 버튼, 입력 필드, 대시보드
- **적용 라이브러리**: Material-UI (React), Vuetify (Vue)

### 2. **Ant Design** (https://ant.design)

- 엔터프라이즈급 UI 컴포넌트
- **특징**: 깔끔한 디자인, 풍부한 컴포넌트
- **적용 라이브러리**: Ant Design (React)

### 3. **Chakra UI** (https://chakra-ui.com)

- 모던하고 접근성 높은 컴포넌트
- **특징**: 커스터마이징 용이, 다크모드 지원
- **적용 라이브러리**: Chakra UI (React)

### 4. **Tailwind CSS** (https://tailwindcss.com)

- 유틸리티 우선 CSS 프레임워크
- **특징**: 빠른 개발, 커스터마이징 자유도 높음
- **예시 사이트**: https://tailwindui.com

## 🎯 AI Adviser 서비스에 특화된 디자인 참고 포인트

### 1. **대시보드 디자인**

- 대학원 목록 카드형 레이아웃
- 교수님 프로필 카드 (사진, 이름, 연구 분야, 적합도 점수)
- **참고**: Notion 대시보드, Linear 대시보드

### 2. **매칭 결과 표시**

- 적합도 점수 시각화 (게이지, 프로그레스 바, 원형 차트)
- Indicator별 점수 비교 (라더 차트, 바 차트)
- **참고**: LinkedIn 매칭, Tinder 스와이프 카드

### 3. **채팅 인터페이스**

- 메시지 버블 디자인
- 타이핑 인디케이터
- 메시지 시간 표시
- **참고**: ChatGPT, Discord, Slack

### 4. **리포트/분석 화면**

- 리포트 카드 레이아웃
- 차트 및 그래프 (Chart.js, Recharts)
- **참고**: Google Analytics, Mixpanel

### 5. **이메일 작성 화면**

- 텍스트 에디터 (Rich Text Editor)
- 미리보기 기능
- **참고**: Gmail, Notion 에디터
