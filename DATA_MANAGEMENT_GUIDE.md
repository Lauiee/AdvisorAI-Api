### 방법 2: Python 코드로 직접 추가

Python 스크립트를 작성하여 직접 추가할 수도 있습니다:

```python
from add_data import add_graduate_school, add_professor

# 대학원 추가
school = add_graduate_school(
    name="서강대학교 기술경영전문대학원",
    description="기술경영 분야의 전문 대학원"
)

# 교수님 추가
professor = add_professor(
    professor_id="prof_001",
    name="박현규",
    graduate_school_id=1,  # 위에서 추가한 대학원의 ID
    major="기술경영학",
    research_fields="디지털 기업가 정신, 전략기획도구, 정성연구방법론",
    introduction="박현규 교수는 University of Cambridge에서 기술경영학 박사학위를 취득하고 현재 서강대학교 기술경영전문대학원 부교수로 재직 중이다.",
    education="Ph.D. Technology and Innovation Management, University of Cambridge",
    career="2021.03~현재 서강대학교 기술경영전문대학원 교수",
    courses="기술혁신론, 기술로드매핑 이론과 실습, 정성연구와 사례개발"
)
```

## 📋 입력 필드 설명

### 대학원 정보
- **이름** (필수): 대학원 이름
- **설명** (선택): 대학원에 대한 설명

### 교수님 정보
- **교수님 ID** (필수): 고유 식별자 (예: prof_001, prof_002)
- **이름** (필수): 교수님 성함
- **대학원 ID** (필수): 소속 대학원의 ID (먼저 대학원을 추가해야 함)
- **전공** (선택): 전공 분야
- **연구 분야** (선택): 주요 연구 분야
- **소개** (선택): 교수님 소개글
- **학력** (선택): 학력 정보
- **경력** (선택): 경력 정보
- **담당 과목** (선택): 담당 과목 목록

## ✅ 추가 확인

데이터가 제대로 추가되었는지 확인하려면:

### 1. API로 확인
```bash
# 대학원 목록 조회
curl http://localhost:8000/graduate-schools

# 특정 대학원의 교수님 목록 조회
curl http://localhost:8000/graduate-schools/1/professors
```

### 2. 스크립트로 확인
```bash
python3 add_data.py
# 메뉴에서 "3. 대학원 목록 조회" 선택
```

## 💡 팁

1. **대학원을 먼저 추가하세요**: 교수님을 추가하려면 먼저 해당 대학원이 존재해야 합니다.
2. **교수님 ID는 고유해야 합니다**: 같은 ID를 가진 교수님은 추가할 수 없습니다.
3. **선택사항 필드는 비워둘 수 있습니다**: Enter 키를 눌러 건너뛸 수 있습니다.
